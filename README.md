# rs-base-framework

> Fast interface building for react

## Install

```bash
npm install --save @xdev-asia/x-react-elements
```

## Usage

```jsx
import React, { Component } from 'react'

import { Pages } from '@xdev-asia/x-react-elements'

class Example extends Component {
  render() {
    return <Pages.Login />
  }
}
```
